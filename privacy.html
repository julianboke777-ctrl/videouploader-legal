Privacy Policy — Video Uploader

Effective date: 2026-05-07

1. Overview

Video Uploader (the "Application") is a single-developer automation tool used exclusively by its operator to publish self-produced video content to TikTok creator channels owned and operated by the operator. This Privacy Policy describes the limited personal data the Application processes.

2. Data Controller

The Application is operated by a single individual developer (the "Operator"), residing in Germany. For data-protection inquiries, contact the Operator via the email address registered with the application's TikTok for Developers account.

3. Data Collected and Processed

Data categorySourcePurpose
OAuth access_tokenTikTok Login Kit (Authorization Code Flow)Authentication to the TikTok Content Posting API
OAuth refresh_tokenTikTok Login KitRenewing the access token after expiry
TikTok open_id (pseudonymous user identifier)TikTok Login KitMapping the token to the correct creator account
Video file (self-produced by the Operator)Local file systemUpload via the TikTok API
Caption text (authored by the Operator)Local file systemSubmitted with the video

The Application does not process: data of other TikTok users, analytics or performance data, IP addresses, location data, or device identifiers.

4. Legal Basis (GDPR)

Processing is performed under Art. 6(1)(b) GDPR (performance of a contract — the Operator authorizes upload via the OAuth flow) supplemented by Art. 6(1)(a) GDPR (consent given through OAuth authorization).

5. Storage Location and Retention

OAuth tokens are stored exclusively locally on the Operator's machine in a file with permission 0600. No tokens are stored on any server, cloud service, or third-party processor (other than the TikTok API itself).

  • Access token: up to 24 hours from issuance (per TikTok's specification)
  • Refresh token: up to 365 days from issuance, or until manual revocation

6. Recipients

The sole external recipient of data transmitted via the Application is TikTok Technology Limited (Dublin, Ireland) and, where applicable, affiliated entities of the ByteDance group, in accordance with TikTok's own privacy policy.

7. International Transfers

To the extent TikTok itself transfers data to servers outside the European Economic Area, those transfers are governed by EU Standard Contractual Clauses pursuant to Art. 46(2)(c) GDPR. The Operator does not himself initiate such transfers.

8. Data Subject Rights

The Operator is the only data subject and has full control over the locally-stored tokens, which can be deleted at any time. OAuth authorization can additionally be revoked directly in the TikTok app: Settings → Privacy → Manage apps and websites → Video Uploader → Remove access.

Standard rights under Articles 15-22 GDPR apply. Complaints may be lodged with the supervisory authority of the Operator's federal state (a list is available at bfdi.bund.de/anschriften).

9. Automated Decision-Making

No automated decision-making within the meaning of Art. 22 GDPR and no profiling are performed.

10. Changes

This Privacy Policy may be updated to reflect changes in the Application's behavior or applicable law. The current version is the authoritative one.
